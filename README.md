# Water & Ice

Water & Ice is an online multiplayer mobile game developed using Unity and C#.

## Screenshot

![Gameplay](Frame%20161%20(1).png)

## Features

- Online multiplayer gameplay
- Room creation and joining
- Water and Ice role system
- Freeze and unfreeze mechanics
- Match timer and win conditions

## Technologies Used

- Unity
- C#
- Unity Netcode
- Unity Relay
- Unity Lobby
